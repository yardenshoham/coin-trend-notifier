login

    • Defined in src/controllers/userController.ts:50

    Logins a user to the system.

    Parameters

    • userCredentials: UserLoginDto

      The credentials (email and password) of the user that wants to login.

    • res: Response

    Returns Promise<Response>

    One of the following:

    • A { jwt: string } object containing the json web token with the payload UserJwtPayload, used for authentication of all requests. Status: OK.
    • A { error: string } object if the user is not registered or an incorrect password was provided. Status: UNAUTHORIZED.
